Reading Assignments

Carrano, F.M. and Prichard, J.J., Data Abstraction and Problem Solving with Java: Walls and Mirrors , Addison Wesley, 2006 or 2010 (2nd or 3rd -- either edition). This is a recommended, not required text.

Readings will be from on-line sources and from the recommended text or your equivalent. The recommended text is used because it was required for Comp 182/L. This allows students to reuse the text. It is also on reserve in the library. There is little difference between 2ed and 3ed. The alternate recommended text has a slightly more advanced presentation than the recommended text. Alternate recommended text: Weiss, M.A., Data Structures & Problem Solving using Java, 4th ed, Addison Wesley, 2010.

All Chapter readings are from recommended text (or equivalent).

  • Chapter 4.3, 5.5, 9 -- review Java: ADT, iterators, generics, interfaces
  • Swing, GUI API -- if interested
    • Swing tutorial Getting started with Swing
    • Swing tutorial Using Swing Components Skip the section on using netbeans. You can use any IDE you want (JGrasp, Dr.Java, Eclipse, ...). You don't need netbeans to do any GUI design in the class assignments.
