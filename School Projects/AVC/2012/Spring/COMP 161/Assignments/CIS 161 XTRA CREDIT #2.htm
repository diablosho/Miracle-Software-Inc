Extra Credit Assignment #2

#2 Create an application that finds all 4 digit numbers (those would be the numbers in the range 1000-9999) whose value is equal to the sum of each digit raised to the fourth power. MEANINGFUL output should be sent to the screen.

For example, if 1^4 + 2^4 + 3^4 + 4^4 = 1234, then your program would print 1234 as one of the solutions that meets the criteria.

Turn in your source code and several sample runs.
